MNIST Datasets¶

O banco de dados MNIST (do ingls, Modified National Institute of Standards and Technology database) um grande banco de dados de dgitos manuscritos que comumente usado para o treinamento de vrios sistemas de processamento de imagens. Este banco de dados tambm amplamente utilizado para treinamento e testes no campo da machine learning. Foi criado misturando as amostras dos conjuntos de dados originais do NIST.

Os criadores acharam que, como o conjunto de dados de treinamento do NIST foi obtido de funcionrios do American Census Bureau, enquanto o conjunto de dados de teste foi coletado de estudantes americanos do ensino mdio, no era adequado para experimentos de aprendizado de mquina. Alm disso, as imagens em preto e branco do NIST foram normalizadas para caber em uma caixa delimitadora de 28x28 pixels e anti-alias, o que introduziu nveis de escala de cinza.

O MNIST contm 60.000 imagens de treinamento e 10.000 imagens de teste. Metade do conjunto de treinamento e metade do conjunto de testes foram retirados do conjunto de dados de treinamento do NIST, enquanto a outra metade do conjunto de treinamento e a outra metade do conjunto de testes foram retirados do conjunto de dados de teste do NIST.

Criando uma nova rede neural convolucional (CNN)¶

Uma Rede Neural Convolucional (do ingls, Convolutional Neural Network ou CNN) um algoritmo de Deep Learning que pode captar uma imagem de entrada, atribuir importncia (pesos e vieses que podem ser aprendidos) a vrios aspectos / objetos da imagem e ser capaz de diferenciar um do outro. O pr-processamento exigido em uma CNN muito menor em comparao com outros algoritmos de classificao. Enquanto nos mtodos primitivos os filtros so feitos mo, com treinamento suficiente, as CNNs tm a capacidade de aprender esses filtros / caractersticas.

A arquitetura de uma CNN anloga quela do padro de conectividade de neurnios no crebro humano e foi inspirada na organizao do Visual Cortex. Os neurnios individuais respondem a estmulos apenas em uma regio restrita do campo visual conhecida como Campo Receptivo. Uma coleo desses campos se sobrepe para cobrir toda a rea visual.

Uma nova rede convolucional ser criada com as caractersticas definidas, conforme cdigo a seguir.

Random Forest¶

Os chamados mtodos ensemble, so construdos da mesma forma que algoritmos mais bsicos, como regresso linear, rvore de deciso ou knn, por exemplo, mas possuem uma caracterstica principal que os diferenciam, a combinao de diferentes modelos para se obter um nico resultado. Essa caracterstica torna esses algoritmos mais robustos e complexos, levando a um maior custo computacional que costuma ser acompanhando de melhores resultados.

Normalmente na criao de um modelo, escolhemos o algoritmo que apresenta o melhor desempenho para os dados em questo. Podemos testar diferentes configuraes deste algoritmo escolhido, gerando assim diferentes modelos, mas no fim do processo de machine learning, escolhemos apenas um. Com um mtodo ensemble sero criados vrios modelos diferentes a partir de um algoritmo, mas no escolheremos apenas um para utilizao final, e sim todos.

No algoritmo Random Forest sero criadas vrias rvores de deciso,a floresta que ele cria uma combinao (ensemble) de rvores de deciso, na maioria dos casos treinados com o mtodo de bagging. A idia principal do mtodo de bagging que a combinao dos modelos de aprendizado aumenta o resultado geral.

Uma grande vantagem do algoritmo de florestas aleatrias que ele pode ser utilizado tanto para tarefas de classificao quanto para regresso, o que representa a maioria dos sistemas de aprendizagem de mquina atuais.

def hyper_RF(hyper_space):

                                criterion = hyper_space['criterion'],

    if (acc < best_score):

    return {'loss': acc, 'status': STATUS_OK }
